
# Service Runner

一个简单的集成测试框架，基于:

1. pytest
2. requests
3. jinja2
4. pytest-allure

## 初始化项目

- 安装Python3 环境: 下载PYTHON3，并且安装,设置环境变量
- 初始化 - 安装pipenv,python 虚拟环境

```sh
pip install pipenv
```

- 安装依赖

```sh 
pipenv install
```

- 运行demo看看结果

```sh 
sh <测试文件路径>
```

测试结果会自动打开
